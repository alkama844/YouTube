## ğŸ” Troubleshooting

### Videos won't load?
- **Check internet connection**
- **Refresh page** (Ctrl+R)
- **Clear cache** in Settings
- **Try different quality** setting

### "Video Cannot Be Embedded"?
- Some videos are restricted by creators
- Click **Open in YouTube** to watch in app/browser
- This is a YouTube restriction, not an app issue

### Fullscreen not working?
- Make sure browser allows fullscreen
- Try clicking **â›¶ Full** button
- Press `F` key as alternative
- Check browser permissions

### Theme not applying?
- Refresh the page
- Clear browser cache
- Try different browser
- Check localStorage is enabled

### Stats not showing?
- Watch a few videos first
- Stats update after video plays
- Check History tab has entries
